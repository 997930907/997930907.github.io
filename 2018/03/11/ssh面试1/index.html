By 余明星 at: 2018-03-11
Java

SSH面试总结之一

1. 为什么使用SSH框架?

  • Struts2：基于MVC设计模式的开源框架，有丰富的标签库。
  • Hibernate：开源的ORM框架（对象关系映射框架）。
  • Spring：基于IOC、AOP组件的轻量级开源框架。

2. 为Struts2工作原理

  1. 浏览器发送Http请求到web服务器
  2. 执行Struts2核心过滤器StrutsPrepareAndExecuteFileter( 在服务器启动的时候创建，执行init()方法来加载配置文件，执行doFilter()方法，最后执行destory()方法 )。
  3. 加载Struts配置文件信息，找到相应的action类，并实例化。
  4. 执行各类拦截器和aciton中对应的方法。
  5. 从配置文件中找到返回结果
  6. 转发到相应的页面。

为什么用Spring框架

Sptirng是一个轻量级的控制反转、面向切面、面向接口和事务管理的开源框架。
IOC：控制反转，交给Spring配置来创建对象和注入依赖对象。好处是松耦合。

底层原理

  1. xml配置文件创建对象类
  2. 创建工厂类，通过dom4j解析xml文件，通过反射创建类。

Class clazz = Class.forName(classValue);
UserService service = clazz.newInstance();
return service;

AOP：面向切面编程，扩展功能不修改源代码实现。采用横向抽取机制。
底层原理：动态代理实现。
